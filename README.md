# Procedural Space Miner

A small 3D browser game built with Three.js and Vite. Fly a ship through space, mine procedural asteroids with an energy-limited laser, collect gems, and upgrade speed, laser strength, and laser duration.

## Run Locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite.

## Controls

- Move: `WASD` or arrow keys
- Aim: mouse or pointer
- Fire mining laser: hold click, `Space`, or `F`
- Spend collected gems with the upgrade buttons

All visuals are procedural; there are no image or model assets to download.
